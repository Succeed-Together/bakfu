bakfu

bakfu is a Python module used to collect tools for machine learning and natural language processing. It is distributed under the 3-Clause BSD license.

Example

import bakfu
baf = bakfu.Chain().load('data.simple',data)
baf.process('sklearn.vectorize')
baf.process('cluster.ward')
result = baf.get('result')

Dependencies

bakfu requires scikit-learn (http://scikit-learn.org) (and thus NumPy and SciPy).
